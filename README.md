# Razor Pages Demo

Đây là một dự án Razor page đơn giản phục vụ mục đích học tập.

## Cách cài đặt và chạy dự án

## Cài đặt

Clone repo:
   ```sh
   git clone https://github.com/phandinhphu/LearnRazorPage.git
   ```
## Chạy dự án

Khởi chạy ứng dụng với lệnh:
```sh
dotnet run
```
Hoặc chạy ứng dụng với lệnh:
```sh
dotnet run watch
```
